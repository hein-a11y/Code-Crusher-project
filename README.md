# Code-Crusher-project
GG store 開発 

login 機能とか作るとき, customer 情報をsessionに登録ため、 $_session['customer'] に共通したいので名前を customer にしてください。
cart 機能とか作るとき, product 情報をsessionに登録ため、 $_session['product'] に共通したいので名前を product にしてください。
